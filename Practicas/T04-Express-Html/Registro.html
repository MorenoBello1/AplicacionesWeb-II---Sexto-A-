Entidad Registro

Esta entidad transaccional se almacenan los debidos registros de los pacientes, en la que hace uso de relaciones con las iDs en Plato y Paciente

ATRIBUTOS

ID_REGISTRO: En este atributo se almacenara la Id unica de cada uno de los Registros
ID_PACIENTE: En este atributo se almacenara la Id unica de cada uno de los pacientes, siendo una id con relacion a la entidad paciente.
ID_PLATO: En este atributo se almacenara la Id unica de los platos, siendo una id con relacion a la entidad plato.
FECHA: En este atributo se almacenara las fechas en la que sucede el registo.
HORA: En este atributo se almacenara las hora en la que sucede el registo.
NUMERO_DE_CALORIAS_CONSUMIDAS: En este atributo se almacenan las calorias que consumio x paciente.
NUMERO_PORCIONES: Este atributo se almacena las porciones totales consumida por el paciente en ese registro
