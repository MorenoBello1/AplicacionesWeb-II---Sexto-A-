Entidad Paciente

Esta entidad Almacenara los paciente con sus respectivos nombres y datos para posteriormente ser registrados dentro del sistema

ATRIBUTOS

ID_PACIENTE: En este atributo se almacenara la Id unica de cada uno de los pacientes
NOMBRE: En este atributo se almacenara los nombres respectivos del paciente.
IDENTIFICACION: En este atributo se almacenan las cedulas o documentos con el cual se identificara ejemplo: "1314845791"
EDAD: En este atributo se almacenan las edades de cada uno de los pacientes.
ALTURA: Este atributo es necesario por si se desea hacer algun calculo o proceso al paciente, aqui se almacenan la altura del paciente.
